# Echo Mirage

**Copy-first. Proposal-driven. AI-native.**

Echo Mirage is a document workflow where nothing gets overwritten by accident.
Instead of mutating the original, AI makes a copy, proposes a change, and lets the user review it safely.

- `Echo` = copies
- `Mirage` = proposals
- `Operator` = direct human edits
- `Workshop` = AI-made proposal flow
- `Accept` = promote the copy
- `Reject` = discard it

It works for documents, code, voice, and anything else that should stay safe while it changes.

**A better git for humans and AI.**
