# Core Problem

Traditional file systems organize knowledge through:
- folders
- paths
- rigid hierarchies
- static locations

This creates problems:
- files move
- folders drift
- names change
- duplicates appear
- relationships become hidden
- operational meaning becomes fragmented

Paths are temporary.

Operational identity should survive movement.

---

The Semantic Atlas exists to preserve:
- continuity

- contextual topology

The Atlas is not merely a search engine.

- continuity infrastructure
